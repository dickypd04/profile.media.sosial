LinkHub - Final Project (generated)
Files:
 - index.html (reads config/*.json)
 - css/style.css
 - js/main.js
 - config/profile.json (username, photo, bio, brand)
 - config/links.json (array of link objects: label, url, subtitle)
 - config/theme.json (mode: 'dark' or 'light')
 - assets/profile.svg (placeholder avatar)

How to use:
1. Unzip and host on any static host (Netlify, Vercel, GitHub Pages).
2. Replace assets/profile.svg with your photo or update config/profile.json 'photo' to an external URL.
3. Edit config/links.json to change links & labels.

Enjoy!
